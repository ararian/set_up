# set_up
* set_up.ps1を実行することで、開発に必要なソフトウェアを簡単にインストール可能
* インストールするアプリケーションは随時要確認

# Requirement
powershellが実行可能な環境

# Usage
* 管理者として実行すること
* プロジェクトのダウンロード
* powershellの実行権限を一時的に強化
* スクリプトの実行

```bash
git clone https://github.com/ararian/set_up.git
cd set_up
Set-ExecutionPolicy Bypass -Scope Process -Force

```
